# Telegram Alert Bot

Este bot envia mensagens automaticamente para um grupo/usuário no Telegram via API.

## Como usar

1. Cria um ficheiro .env com:
BOT_TOKEN=teu_token_aqui  
CHAT_ID=teu_chat_id_aqui

2. Corre localmente com:
pip install -r requirements.txt  
python main.py

3. Faz deploy gratuito no Render (usa o Procfile).
